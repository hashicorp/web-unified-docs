# Add a code server - GitHub

HCP Vault Radar allows you to connect to any organization on [GitHub Cloud](#add-github-cloud) and [GitHub Enterprise](#add-github-enterprise). 

## Add GitHub Cloud

### Prerequisites

- Organization URL
- Personal Access Token (with the right set of scopes)

## Create a connection with GitHub Cloud

1. Select **Settings/Data Sources**.

1. Click on the **GitHub Cloud** button.

1. Enter the organization url in the **Organization URL** text field.

1. Enter the personal acces token in the **Personal Access Token** text field.

  ![Configure GitHub Cloud](/img/docs/vault-radar/github-pat-scopes.png)

1. Click **Next**.

## Add GitHub Enterprise

### Prerequisites

- Domain name of your instance
- Organization URL
- Personal Access Token (with the right set of scopes)

## Create a connection with GitHub Enterprise

1. Select **Settings/Data Sources**.

1. Click on the **GitHub Enterprise** button.

1. Enter the domain url in the **Domain name** text field.

1. Enter the organization url in the **Organization URL** text field.

1. Enter the personal acces token in the **Personal Access Token** text field.

  ![Configure GitHub Enterprise](/img/docs/vault-radar/github-pat-scopes.png)

1. Click **Next**.

## Add repositories to Vault Radar

Once the connection had been established, you can select which repositories you
want to install by selecting a group or all.

<Note> 

HCP Vault Radar has a limit of 100 repositories per connection. If you have more
then 100 repositories, HCP Vault Radar will select the 100 repositories with the
most recent activity.

</Note>
