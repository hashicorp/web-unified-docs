# LLM Skill Files

This directory contains skill files for use with IBM Bob or other LLM tools.

These skill files are intended for document authors to make self-review faster and easier.

## Recommended Usage Pattern

To run a skill file in IBM Bob, prompt Bob to read the skill file and run it.  `Read the skill in ./docs/skills/basic-english-check and run it.`  

We recommend running the skills on new or updated content in the following order, fixing any issues found after each run:
1. Basic English check for language and mechanics (`basic-english-check`)
2. User profile and context check (`branch-user-context-check`) to make sure your content is written at a level the intended reader can follow
3. "Top 12" style guide issues (`style-guide-top-12-check`) to find the most common style guide violations
4. Full style guide check (`style-guide-full-check`) for a comprehensive automated review.

Please report your experience in #team-education.

## Skill List

- basic-english-check/skill.md - Performs basic English writing mechanics checks on a file, including spelling, punctuation and spacing, subject-verb agreement, passive voice detection, and other grammar rules. Reports findings and potential fixes.
- branch-user-context-check/skill.md - Analyzes all files changed in a branch to compare the intended reader profile with the context and background knowledge needed to understand the content.
- create-style-guide-checklist/skill.md - Generates a checklist from the latest version of the HashiCorp style guide located in /docs/style-guide. Can create either a FULL checklist (one-to-one mapping with style guide rules) or a SUMMARIZED checklist (combining similar rules). Includes traceability checks to ensure all style guide rules are represented. Not likely to be run often.
- style-guide-full-check/skill.md - Comprehensive style guide compliance checker that reviews a markdown file against the complete HashiCorp style guide. 
- style-guide-top-12-check/skill.md - Quick style guide checker that focuses on the "Top 12" most common and important HashiCorp style guide issues. Reads the Top 12 guidelines from ./docs/style-guide/top-12.md and evaluates a target file for violations of these key rules.

## Utility Files

- full_style_guide_checklist.md - A cached version of the full checklist generated by `create-style-guide-checklist`.  Having a cached version available will save tokens for users.
