## Deprecations

1. The following Terraform Enterprise admin CLI subcommands are deprecated. HashiCorp plans to remove these commands in the next major release:
- `tfectl db last-applied-migration`
- `tfectl db backup`
- `tfectl db reindex`
- `tfectl db restore`


## Highlights
