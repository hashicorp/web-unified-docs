# Add Open Policy Agent (OPA) Tool Versions

Navigate to the **OPA Versions**
- Click on the `Add OPA version` button.
- In the `Version` text box, enter `0.44.0`.
- Enable **Enable this version** check box.
- In the `URL` text box, enter `https://github.com/open-policy-agent/opa/releases/download/v0.44.0/opa_linux_amd64_static`.
- In the `SHA256 Checksum` text box, enter `5ddb21d3fcfca130a47a42e730c05f055c68af6c1b37465879f6c59b10527eae`.
- Click on the `Add OPA version` button.

All the details can also be found in OPA Github repo releases page: `https://github.com/open-policy-agent/opa/releases`
