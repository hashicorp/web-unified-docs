# Terraform Enterprise Flexible Deployment Options administration

Terraform Enterprise Flexible Deployment Options is a software distribution that manages a private instance of the Terraform Cloud application on either the Docker Runtime, or Kubernetes.

For information on how to manage and support your deployment:

- **[Admin CLI](/terraform/enterprise/flexible-deployments-beta/admin/admin-cli/admin-cli):** This section contains information about how to access and use the tfectl command line interface to administer the Terraform Enterprise server.
- **[Backup and Restore](/terraform/enterprise/flexible-deployments-beta/admin/admin-cli/backup-restore.mdx):** Instructions on how to use the native backup and restore API.
- **[Support](/terraform/enterprise/flexible-deployments-beta/admin/support):** This section contains information on how to receive support for the Flexible Deployment Options Beta.
