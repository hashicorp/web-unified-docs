Requests to the [`/api/v1/ping`](/terraform/enterprise/api-docs/ping) and [`/api/v1/usage/bundle`](/terraform/enterprise/api-docs/usage-bundle) endpoints must be authenticated with a bearer token generated specifically for them using the `tfectl admin api-token generate` command. For more information on the token creation, and management, refer to the [tfectl documentation](/terraform/enterprise/deploy/reference/cli).

Use the HTTP Header `Authorization` with the value `Bearer <token>`.
