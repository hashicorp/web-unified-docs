Make sure that your network allows HTTPS egress on port 443 to
`https://reporting.hashicorp.services` by allow-listing the following IP addresses:

- 100.20.70.12
- 35.166.5.222
- 23.95.85.111
- 44.215.244.1

You can verify port access is correctly configured with a curl request:

```shell-session
$ curl 'https://reporting.hashicorp.services/v2/' -d '{}'
```

You should receive an "invalid bundle" response:

```json
{"message": "invalid bundle"}
```

The server returns an invalid bundle response because it received the request successfully, but could not parse the empty dataset provided.
