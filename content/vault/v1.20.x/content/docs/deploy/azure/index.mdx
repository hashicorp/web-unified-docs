# Run Vault on Microsoft Azure

You can deploy Vault on Microsoft Azure in several ways:

- Use a [HCP Vault Dedicated tier](https://portal.cloud.hashicorp.com) deployed on Azure.
- Deploy Vault Enterprise on Microsoft Azure with the official HashiCorp
[Azure Marketplace offering of Vault Enterprise](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/hashicorp-4665790.vault-azure-saas?tab=overview).
- Deploy Vault to Azure Kubernetes Service (AKS) with the official HashiCorp [Vault Helm chart](/vault/docs/deploy/kubernetes/helm).
- Configure and deploy Vault manually on an Azure VM instance.
