# Run Vault on Microsoft Azure

A number of ways to deploying Vault on Microsoft Azure are available:

- [HCP Vault Dedicated tier](https://portal.cloud.hashicorp.com) deployed on Azure.
- Vault Enterprise on Microsoft Azure with the official HashiCorp
[Azure Marketplace offering of Vault Enterprise](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/hashicorp-4665790.vault-azure-saas?tab=overview).
- AKS with the official HashiCorp [Vault Helm chart](/vault/docs/deploy/kubernetes/helm).
- Manual deployment on an Azure VM instance.
