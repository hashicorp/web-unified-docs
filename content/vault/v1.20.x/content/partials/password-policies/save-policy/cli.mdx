Use `vault write` with the
[`/sys/policies/password/{policy_name}`](/vault/api-docs/system/policies-password#create-update-password-policy)
path and your policy file to create a new password policy:

```shell-session
$ vault write /sys/policies/password/<policy_name> \
  policy=<policy_configuration_file>
```

For example:

<CodeBlockConfig hideClipboard="true">

```shell-session
$ vault write /sys/policies/password/racf-passphrase \
  policy=racf-policy.hcl
```

</CodeBlockConfig>
