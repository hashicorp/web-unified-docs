Make a `POST` call to
[`/sys/leases/renew`](/vault/api-docs/system/leases#renew-lease)
with the full lease ID of the service account to extend the lease on a service
account:

```shell-session
$ curl                                                \
  --request POST                                      \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  --data '{"lease_id": "<mount_path>/library/<set_name>/check-out/<lease_token>"}'  \
  ${VAULT_ADDR}/v1/sys/leases/renew
```
