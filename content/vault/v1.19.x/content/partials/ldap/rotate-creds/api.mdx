Make a `POST` call to
[`/{mount_path}/rotate-role/{role_name}`](/vault/api-docs/secret/ldap#manually-rotate-static-role-password)
to rotate the credential password:

```shell-session
$ curl                                                \
  --request POST                                      \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  ${VAULT_ADDR}/v1/<mount_path>/rotate-role/<role_name>
```
