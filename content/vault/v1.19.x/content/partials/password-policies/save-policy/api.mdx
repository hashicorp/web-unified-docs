Make a `POST` call to
[`/sys/policies/password/{policy_name}`](/vault/api-docs/system/policies-password#create-update-password-policy)
with your your policy file to create a new password policy:

```shell-session
$ curl                                                \
  --request POST                                      \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  --data {"policy": @<policy_configuration_file>}     \
  ${VAULT_ADDR}/v1/sys/policies/password/<policy_name>
```
