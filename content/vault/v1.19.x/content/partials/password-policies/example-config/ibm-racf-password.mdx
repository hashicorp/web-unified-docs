The following password policy adheres to the password policy in IBM RACF by 
requiring:

- 8 characters.
- at least one lowercase character.
- at least one uppercase character.
- at least one number.

```hcl
length = 8

rule "charset" {
  charset = "abcdefghijklmnopqrstuvwxyz"
  min-chars = 1
}

rule "charset" {
  charset = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
  min-chars = 1
}

rule "charset" {
  charset = "0123456789"
  min-chars = 1
}
```
