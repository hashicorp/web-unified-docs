# Run Vault on Google Cloud Platform (GCP)

You can run Vault on Google Cloud Platform (GCP) in the following ways:

- GKE with the official HashiCorp [Vault Helm chart](/vault/docs/deploy/kubernetes/helm).
- Manual deployment on a Google Compute Engine instance.
