# Oracle database plugin HTTP API

The Oracle database plugin extends the built in Vault `database` secrets engine
to generate database credentials dynamically based on configured roles for your
Oracle database.

## Endpoints 

Name                                                 | Method | Path
---------------------------------------------------- | ------ | ------------------------
[Config](/vault/plugins/databases/oracle/api/config) | `POST` | `/database/config/:name`
