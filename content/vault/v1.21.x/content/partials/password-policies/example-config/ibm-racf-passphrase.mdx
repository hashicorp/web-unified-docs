The following password policy adheres to the passphrase policy in IBM RACF by 
requiring:

- 14 characters.
- at least one lowercase character.
- at least one uppercase character.
- at least one number.
- at least special character from the set [`!`, `@`, `#`, `$`, `%`, `&`].

```hcl
length = 14

rule "charset" {
  charset = "abcdefghijklmnopqrstuvwxyz"
  min-chars = 1
}

rule "charset" {
  charset = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
  min-chars = 1
}

rule "charset" {
  charset = "0123456789"
  min-chars = 1
}

rule "charset" {
  charset = "!@#$%&"
  min-chars = 1
}
```
