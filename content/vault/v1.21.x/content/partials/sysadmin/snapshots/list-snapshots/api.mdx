You can check to see if there is a snapshot currently loaded by querying 
the `/sys/storage/raft/snapshot-load` endpoint:

```shell-session
$ curl                                       \
   --request LIST                            \
   --header "X-Vault-Token: ${VAULT_TOKEN}"  \
    ${VAULT_ADDR}/v1/sys/storage/raft/snapshot-load
```
