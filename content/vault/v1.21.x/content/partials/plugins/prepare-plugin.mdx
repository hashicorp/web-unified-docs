You must place the plugin in your configured `plugin_directory`. Vault expects
specific path structures based on the license and format.

**Important for Enterprise Plugins**: The `<name>` used during registration
must match the plugin name as it appears on the
[HashiCorp Releases page](https://releases.hashicorp.com). For example,
`vault-plugin-secrets-keymgmt`.

| Integration | License    | Format | Path structure |
| :--- | :--- | :--- | :--- |
| Official | Enterprise | `.zip` | `<plugin_directory>/<extracted-folder-name>/` |
| Official | Community  | `.zip` | `<plugin_directory>/<extracted-folder-name>/` |
| Official | Community  | binary | `<plugin_directory>/<binary-name>` |
| Community | N/A | binary | `<plugin_directory>/<binary-name>` |
