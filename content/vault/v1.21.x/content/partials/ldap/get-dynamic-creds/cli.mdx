Use `vault read` with the
[`/{mount_path}/creds/{role_name}`](/vault/api-docs/secret/ldap#dynamic-role-passwords)
to fetch the credential information for the dynamic role:

```shell-session
$ vault read <mount_path>/creds/<role_name>
```
