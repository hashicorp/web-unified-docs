Make a `POST` call to
[`/{mount_path}/role/{role_name}`](/vault/api-docs/secret/ldap#create-delete-dynamic-role-configuration)
with your dynamic role configuration, `ldap-role.json`, to create a new
dynamic role entry:

```shell-session
$ curl                                                \
  --request POST                                      \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  --data @ldap-role.json                              \
  ${VAULT_ADDR}/v1/<mount_path>/role/<role_name>
```
