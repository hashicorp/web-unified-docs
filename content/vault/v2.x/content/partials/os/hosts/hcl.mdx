Use the [`vault_os_secret_backend_host`](https://registry.terraform.io/providers/hashicorp/vault/latest/docs/resources/os_secret_backend_host)
resource to configure a host:

```hcl
resource "vault_os_secret_backend_host" "my_box" {
  mount   = vault_os_secret_backend.os.mount
  name    = "my-box"
  address = "box.example.com"
  port    = 22
}
```
