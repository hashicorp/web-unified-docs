1. Configure the parent (superuser) account:

    ```shell-session
    $ vault write os/hosts/my-box/accounts/mgmt-user \
      username="super-user" \
      password="super-secret"
    ```

1. Configure the target account, referencing the parent account:

    ```shell-session
    $ vault write os/hosts/my-box/accounts/app-user \
      username="user-1" \
      password="bigsecret" \
      parent_ref="mgmt-user"
    ```
