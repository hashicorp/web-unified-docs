Use the [`vault_os_secret_backend_account`](https://registry.terraform.io/providers/hashicorp/vault/latest/docs/resources/os_secret_backend_account)
resource to configure the account:

```hcl
resource "vault_os_secret_backend_account" "app_user" {
  mount       = vault_os_secret_backend.os.mount
  host        = vault_os_secret_backend_host.my_box.name
  name        = "app-user"
  username    = "user-1"
  password_wo = "bigsecret"
}
```
