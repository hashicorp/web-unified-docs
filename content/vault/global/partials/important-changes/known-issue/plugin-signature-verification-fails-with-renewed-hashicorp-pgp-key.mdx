### Plugin signature verification fails with renewed HashiCorp PGP key ((#plugin-signature-verification-renewed-pgp-key)) <EnterpriseAlert inline="true" />

| Change      | Status | Vault edition | Affected version | Fixed version |
| ----------- | ------ | ------------- | ---------------- | ------------- |
| Known issue | Open   | Enterprise    | 2.0.0            | No            |

#### Issue

Vault Enterprise can fail to verify external enterprise plugins signed with the
renewed HashiCorp PGP signing key.

Affected workflows can return errors similar to the following:

```text
openpgp: key expired
Signature Verification Error: Invalid signature caused by openpgp: key expired
```

#### Workaround

No workaround is available.
