#### Issue

Vault Enterprise fails to verify external enterprise plugins released since
April 21, 2026 that are signed with the renewed HashiCorp PGP signing key.

This issue only affects new plugin registration for Enterprise plugins released since April 21, 2026.
Existing registered plugins will not be affected.

Affected workflows can return errors similar to the following:

```text
openpgp: key expired
Signature Verification Error: Invalid signature caused by openpgp: key expired
```

#### Workaround

We do not currently have a workaround.
