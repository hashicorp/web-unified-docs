Make a `POST` call to
[`/{mount_path}/config`](/vault/api-docs/secret/ldap#configuration-management)
to apply your configuration file, `ldap-config.json`:

```shell-session
$ curl                                                \
  --request POST                                      \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  --data @ldap.json                                   \
  ${VAULT_ADDR}/v1/<mount_path>/config
```
