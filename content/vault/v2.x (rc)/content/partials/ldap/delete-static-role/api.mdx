Make a `DELETE` call to
[`/{mount_path}/static-role`](/vault/api-docs/secret/ldap#static-roles)
with your static role configuration, `ldap-role.json`, to delete the static
role:

```shell-session
$ curl                                                \
  --request DELETE                                    \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  --data @ldap-role.json                              \
  ${VAULT_ADDR}/v1/<mount_path>/static-role
```
