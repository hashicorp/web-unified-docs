Make a `GET` call to
[`/{mount_path}/creds/{role_name}`](/vault/api-docs/secret/ldap#dynamic-role-passwords)
to fetch the credential information for the dynamic role:

```shell-session
$ curl                                                \
  --request GET                                       \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  ${VAULT_ADDR}/v1/<mount_path>/creds/<role_name>
```
