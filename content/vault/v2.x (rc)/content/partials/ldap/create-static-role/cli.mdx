Use `vault write` with the
[`/{mount_path}/static-role`](/vault/api-docs/secret/ldap#static-roles)
path and your static role configuration, `ldap-role.json`, to create a new
static role:

```shell-session
$ vault write <mount_path>/static-role @ldap-role.json
```

For example:

<CodeBlockConfig hideClipboard>

```shell-session
$ vault write devcreds/static-role @ldap-role.json
```
