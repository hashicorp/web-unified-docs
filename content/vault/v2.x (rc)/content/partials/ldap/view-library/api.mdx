Make a `POST` call to
[`{mount_path}/library/{set_name}`](/vault/api-docs/secret/ldap#library-set-management)
to check the status of your library:

```shell-session
$ curl                                                \
  --request GET                                       \
  --header    "X-Vault-Token: ${VAULT_TOKEN}"         \
  --namespace "X-Vault-Namespace: ${VAULT_NAMESPACE}" \
  ${VAULT_ADDR}/v1/<mount_path>/library/<set_name>
```
