Use `vault read` with the
[`/sys/policies/password/{policy_name}`](/vault/api-docs/system/policies-password#create-update-password-policy)
to read the details of a password policy:

```shell-session
$ vault read /sys/policies/password/<policy_name>
```

For example:

<CodeBlockConfig hideClipboard="true">

```shell-session
$ vault read /sys/policies/password/racf-passphrase
```

</CodeBlockConfig>
