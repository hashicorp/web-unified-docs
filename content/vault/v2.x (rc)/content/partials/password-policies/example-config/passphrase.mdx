The following password policy requires password strings to have:

- 20 characters.
- at least 2 spaces (to encourage three-word phrases).
- at least one lowercase character.
- at least one uppercase character.
- any number of basic punctuation marks.

```hcl
length=20

rule "charset" {
  charset = " "
  min-chars = 2
}

rule "charset" {
  charset = "abcdefghijklmnopqrstuvwxyz"
  min-chars = 1
}

rule "charset" {
  charset = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
  min-chars = 1
}

rule "charset" {
  charset = "!.,?"
  min-chars = 0
}
```
