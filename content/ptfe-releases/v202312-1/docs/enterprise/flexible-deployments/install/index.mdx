# Installation

Terraform Enterprise supports installation on [Docker](/terraform/enterprise/flexible-deployments/install/docker/install) and [Kubernetes](/terraform/enterprise/flexible-deployments/install/kubernetes/install). If you are currently running Terraform Enterprise on Replicated, follow the [migration guide](/terraform/enterprise/replicated/replicated-migration) to move to Flexible Deployment Options. 

Please review the various requirements prior to installation:

- [Shared requirements](/terraform/enterprise/flexible-deployments/install/requirements)

- [Docker requirements](/terraform/enterprise/flexible-deployments/install/docker/requirements)

- [Kubernetes requirements](/terraform/enterprise/flexible-deployments/install/kubernetes/requirements)
