# Terraform Enterprise Flexible Deployment Options administration

Terraform Enterprise Flexible Deployment Options is a software distribution that manages a private instance of the Terraform Cloud application on either the Docker Runtime, or Kubernetes.

For information on how to manage and support your deployment:

- **[Admin CLI](/terraform/enterprise/flexible-deployments/admin/admin-cli/admin-cli):** This section contains information about how to access and use the `tfectl` command line interface to administer the Terraform Enterprise server.
- **[Backup and Restore](/terraform/enterprise/flexible-deployments/admin/admin-cli/backup-restore):** Instructions on how to use the native backup and restore API.
- **[Upgrade](/terraform/enterprise/flexible-deployments/admin/upgrade):** Instructions on how to upgrade Terraform Enterprise Flexible Deployment Options.
