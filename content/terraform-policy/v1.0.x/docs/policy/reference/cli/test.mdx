# `tfpolicy test`

The `tfpolicy test` command runs policy tests to verify that your policies work as expected.

## Usage

```
$ tfpolicy test --policies=</path/to/policies [--tests=/path/to/tests]
```

## Description

Once you have written your policy tests, run them using the `tfpolicy test` command. You can provide path to both the policy files and the test files. If you only specify the path to your policy files, all of the tests in that path will be executed. FIXME: Try this out and clarify.
