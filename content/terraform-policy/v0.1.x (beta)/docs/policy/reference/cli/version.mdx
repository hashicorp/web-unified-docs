# `tfpolicy version`

The `tfpolicy version` command prints out the version of tfpolicy..

## Usage

```
$ tfpolicy version
```

## Description

Use the `tfpolicy version` command to identify the version of the Terraform policy you have installed.

## Examples

Print out the version of the Terraform policy CLI.

```shell-session
$ tfpolicy version
1.0.0
```

## Available options

The `tfpolicy version` command does not support any options.
