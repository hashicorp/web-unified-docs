# `tfpolicy version`

The `tfpolicy version` command prints out the version of the tfpolicy CLI.

@include "partials/beta.mdx"

## Usage

```
$ tfpolicy version
```

## Description

The `tfpolicy version` command identifies the version of the Terraform policy you have installed.

## Examples

Print out the version of the Terraform policy CLI.

```shell-session
$ tfpolicy version
1.0.0
```

## Available options

The `tfpolicy version` command does not support any options.
