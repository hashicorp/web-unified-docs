# `tfpolicy test`

The `tfpolicy test` command runs policy tests to verify that your policies work as expected.

## Usage

```
$ tfpolicy test [--policies=</path/to/policies] [--tests=/path/to/tests]
```

## Description

Once you have written your policy tests, run them using the `tfpolicy test` command. You can provide path to both the policy files and the test files. If no tests argument is provided, the command will load tests from the same directory as the policies. If no policies argument is provided, the command will load policies from the current directory.

## Examples

```shell-session
$ tfpolicy test --policies=./policies --tests=./tests
```

## Available options

You can include the following flags when you run the `tfpolicy test` command:

| Option | Description | Default | Required |
| ------ | ----------- | ------- | -------- |
| `--policies=</path/to/policies>` | Test the policies found in the specified path. | None | No |
| `--tests=</path/to/tests>` | Run the tests found in the specified path. | None | No |
| `--evaluation-stage=<stage> | Test policies for the specified evaluation stage: "setup" or "plan" | None | No |
