# Setup ticketing - ServiceNow

Automatic ticketing of detected issues gives security and developer teams visibility into potential issues so that they can plan, track, and report work. HCP Vault Radar supports ServiceNow as a ticketing tool.

## Prerequisites

- ServiceNow username and password

## Create a ServiceNow connection

1. Select **Settings** from the side navigation. Under the **Integration** menu, click **ServiceNow**.
![Nav start](/img/docs/vault-radar/integrations/servicenow/servicenow-int-1.png)

2. Click the **+ Connection** button.
![Connection start](/img/docs/vault-radar/integrations/servicenow/servicenow-int-2.png)

3. Enter a unique name for the connection.

4. Enter the username of a ServiceNow user with access to the desired account.

5. Enter the password for the username in step 4.

6. Enter the base URL of the account.

7. Click **Test & Save**.
![Connection success](/img/docs/vault-radar/integrations/servicenow/servicenow-int-3.png)

## Create a ServiceNow subscription

1. Click the **+ Subscription** button to begin the creation process.
![Subscription start](/img/docs/vault-radar/integrations/servicenow/servicenow-int-4.png)

2. Choose a name for the subscription as well as the saved filters you would like the subscription to be based on.

3. Choose the connection for the subscription output.

4. (Optional) Enter any [custom field](/hcp/docs/vault-radar/get-started/set-up-ticketing/servicenow/custom-fields) you wish to supply.

5. Click **Test & Save**.
![Subscription complete](/img/docs/vault-radar/integrations/servicenow/servicenow-int-5.png)

HCP Vault Radar events matching your filters will create tickets in the specified ServiceNow project.
