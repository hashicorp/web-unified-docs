# Create custom expressions

HCP Vault Radar custom expressions allows you to receive events when the regular expressions defined by you are matched in resource files. The chosen description also appears in the events list. In general, custom expressions are used to define new secret types, PII information or non-inclusive language.


## Custom expressions

1. Select **Settings/Custom Expressions**.

1. Click Add Regex button.

1. Select a valid category, provide a description, a valid regex and an example for your future reference.

1. Click Save.

  ![Custom Expressions](/img/docs/vault-radar/custom-expressions.png)

You can also edit or delete a defined custom expression. The updated custom expressions do not impact the previous scan results but affect future scans. When you run a reconciliation scan, on-demand scan, PR webhook scan, or similar, you can view events based on the updated regex pattern.
