# Add a code server - BitBucket

HCP Vault Radar allows you to connect to any organization on [BitBucket Cloud](#add-bitbucket-cloud) and [BitBucket Server](#add-bitbucket-server). 

## Add BitBucket Cloud

### Prerequisites

- Workspace name
- Bitbucket account name
- App password with the right set of scopes

## Create a connection with BitBucket Cloud

1. Select **Settings/Data Sources**.

1. Click on the **BitBucket Cloud** button.

1. Enter the workspace in the **Workspace name** text field.

1. Enter the Bitbucket account name and app password value.

  ![Configure BitBucket Cloud](/img/docs/vault-radar/app-password-scopes.png)

1. Click **Next**.

## Add BitBucket server

### Prerequisites

- Domain name for your instance
- Project key
- Bitbucket account name
- App password  (with the right set of scopes)

## Create a connection with BitBucket Cloud

1. Select **Settings/Data Sources**.

1. Click on the **BitBucket Server** button.

1. Enter the workspace in the **Workspace name** text field.

1. Enter the project key in the **Project key** text field.

1. Enter the Bitbucket account name and app password value.

  ![Configure BitBucket Cloud](/img/docs/vault-radar/app-password-scopes.png)

1. Click **Next**.


## Add repositories to Vault Radar

Once the connection had been established, you can select which repositories you
want to install by selecting a group or all.

<Note> 

HCP Vault Radar has a limit of 100 repositories per connection. If you have more
then 100 repositories, HCP Vault Radar will select the 100 repositories with the
most recent activity.

</Note>
