# Set up alerting - Microsoft Teams

You can integrate with Microsoft Teams to subscribe events you wish to monitor.

## Prerequisites

- Generate a [Microsoft Teams channel Incoming WebHook](https://learn.microsoft.com/en-us/microsoftteams/platform/webhooks-and-connectors/how-to/add-incoming-webhook?tabs=dotnet#create-incoming-webhooks-1)

## Create a Microsoft Teams connection

1. Select **Settings** from the side navigation. Under the **Integration** menu, click **Microsoft Teams**.
![Nav teams](/img/docs/vault-radar/integrations/teams/teams-int-1.png)

2. Click the **+ Connection** button.
![Click connection](/img/docs/vault-radar/integrations/teams/teams-int-2.png)

3. Copy the **Incoming WebHook URL** generated in [Microsoft Teams App](https://learn.microsoft.com/en-us/microsoftteams/platform/webhooks-and-connectors/how-to/add-incoming-webhook?tabs=dotnet#create-incoming-webhooks-1).

4. Paste the **Incoming WebHook URL** into the token input and click **Test & Save**.
![Submit connection](/img/docs/vault-radar/integrations/teams/teams-int-3.png)


## Create a Microsoft Teams subscription

1. Click the **+ Subscription** button to begin the creation process.
![Click subscription](/img/docs/vault-radar/integrations/teams/teams-int-4.png)

2. Choose a name for the subscription as well as the saved filters you would like the subscription to be based on.

3. Choose the connection for the subscription output.

4. Click **Test & Save**.
![Submit subscription](/img/docs/vault-radar/integrations/teams/teams-int-5.png)

Hashicorp Vault Radar events matching your filters will be sent to your specified channel in Microsoft Teams.
