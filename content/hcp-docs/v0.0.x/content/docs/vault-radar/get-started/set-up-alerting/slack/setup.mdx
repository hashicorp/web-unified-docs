# Set up alerting - Slack

You can integrate with Slack to subscribe events you wish to monitor.

## Prerequisites

- Create a [Hashicorp Vault Radar Slack App](/TODO/) and install to your workspace.

## Create a Slack connection

1. Select **Settings** from the side navigation. Under the **Integration** menu, click **Slack**.
![Nav slack](/img/docs/vault-radar/integrations/slack/slack-int-1.png)

2. Click the **+ Connection** button.
![Click connection](/img/docs/vault-radar/integrations/slack/slack-int-2.png)

3. Visit the installations page in your [Hashicorp Vault Radar Slack App](https://api.slack.com/apps/) and copy the bot token presented on the screen.

4. Paste the bot token into the token input and click Test & Save.
![Submit subscription](/img/docs/vault-radar/integrations/slack/slack-int-3.png)


## Create a Slack subscription

1. Click the **+ Subscription** button to begin the creation process.
![Click subscription](/img/docs/vault-radar/integrations/slack/slack-int-4.png)

2. Choose a name for the subscription as well as the saved filters you would like the subscription to be based on.

3. Choose the connection for the subscription output and the Slack channel.

4. Click **Test & Save**.
![Submit subscription](/img/docs/vault-radar/integrations/slack/slack-int-5.png)

Hashicorp Vault Radar events matching your filters will be sent to your specified channel in Slack.
