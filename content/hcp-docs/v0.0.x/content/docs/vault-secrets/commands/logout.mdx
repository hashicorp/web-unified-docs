# logout 

Command: `vlt logout` 

Use the `logout` command to log users out of the HCP Vault Secrets web services.


### Examples

The logout command will remove the validated authentication token from your machine, 
but it does not invalidate the web browser session.

```shell-session
$ vlt logout
Successfully logged out
```
