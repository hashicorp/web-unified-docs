# Create an app

You can create new apps from [the HCP Portal](https://portal.cloud.hashicorp.com/), HCP CLI, or HCP API.

## Create first app

1. Open a browser and navigate to the [HCP
   Portal](https://portal.cloud.hashicorp.com/).

1. Log in with an HCP IAM user with the [HCP contributor
   role](/hcp/docs/vault-secrets/get-started/configure-user-permissions) or
   higher.

1. Click on **Vault Secrets** from the left navigation sidebar, or **View Vault
   Secrets** from the card on the dashboard.
   ![HCP Portal with links to access HCP Vault Secrets from the side navigation
   or main dashboard
   highlighted](/img/docs/vault-secrets/ui-access-vault-secrets-portal.png)

1. If no applications exist, you can choose **Create sample app** which will
   create an app with 4 example secrets, or **Create first app** which create an
   app with no example secrets.

   Click **Create first app**.

1. Enter an **App name**. The name may only contain letters, numbers, or
   hyphens. They also must start with a letter and contain 3-36 characters.

1. Enter a **Description**.

1. Click **Create App**.

   You can now add or import secrets.

## Create additional apps

1. From the HCP Vault Secrets **Overview** page, click **Apps**.

1. Click **Create new app**.

1. Enter an **App name**. The name may only contain letters, numbers, or
   hyphens. They also must start with a letter and contain 3-36 characters.

1. Enter a **Description**.

1. Click **Create App**.
