# Access HCP Vault Secrets UI

HCP Vault secrets is accessible from [the HCP Portal](https://portal.cloud.hashicorp.com/), HCP CLI, and HCP API.

1. Open a browser and navigate to the [HCP
   Portal](https://portal.cloud.hashicorp.com/).

1. Log in with an HCP IAM user with [permission to HCP Vault
   Secrets](/hcp/docs/vault-secrets/get-started/configure-user-permissions).

1. Click on **Vault Secrets** from the left navigation sidebar, or **View Vault
   Secrets** from the card on the dashboard.
   ![HCP Portal with links to access HCP Vault Secrets from the side navigation
   or main dashboard
   highlighted](/img/docs/vault-secrets/ui-access-vault-secrets-portal.png)
