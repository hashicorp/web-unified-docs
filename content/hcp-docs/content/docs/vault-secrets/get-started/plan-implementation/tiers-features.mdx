# HCP Vault Secrets tiers and features

HCP Vault Secrets has a Free, Essentials, and Standard tier available. Sign up [here](https://www.hashicorp.com/cloud) for a free account to get started.
Below is a comparison of the features available in each tier.

## Tier comparison

Below is a comparison of the available features and usage limits across Free, Essentials, and Standard tier.

| Features              | Description                                    | Free tier          | Essentials tier      | Standard tier          |
|-----------------------|------------------------------------------------|--------------------|--------------------|--------------------|
| Applications          | The total # of applications                    | 25                 | 1000               | 10000              |
| Static secrets        | The total # of secrets created                 | 25                 | 2500               | 25000              |
| Auto-rotating secrets | The total # of auto-rotating secrets created   | **Standard tier only** | **Standard tier only** | 5000               |
| Dynamic secrets       | The total # of dynamic       secrets created   | **Standard tier only** | **Standard tier only** | 5000               |
| Secret versions       | The total # of versions per secret             | 5                  | 50                 | 50                 |
| Secret syncs          | The total # of active secret sync destinations | 5                  | 200                | 2000               |
| Support               | The level of support provided by Hashicorp     | Bronze             | Starts with Silver | Starts with Silver |

**Secret access API operations:** Any operation that fetches sensitive secret values is counted as a secret access operation. This includes:
- [OpenAppSecrets](/hcp/api-docs/vault-secrets#OpenAppSecrets)
- [OpenAppSecret](/hcp/api-docs/vault-secrets#OpenAppSecret)
- [OpenAppSecretVersion](/hcp/api-docs/vault-secrets#OpenAppSecretVersion)
- [ListOpenAppSecretVersions](/hcp/api-docs/vault-secrets#ListOpenAppSecretVersions)

## API rate and resource limits

### API rate limits

Below rate limits are applicable to Essentials and Standard tier.

| Operation                    | Limit                |
| ---------------------------- | -------------------- |
| Secret access API operations | 6000 requests/minute |
| Other API operations         | 2000 requests/minute |

### Resource limits

For certain resources which are part of a parent-child relationship within the HCP Vault Secrets resource hierarchy, there are system limits for how many child resources can exist within its parent resource. These apply to all tiers.

For information on additional resource limits within HashiCorp Cloud Platform, please refer to [service quotas](/hcp/docs/hcp/admin/support#service-quotas).

Below resource limits are applicable to Essentials and Standard tier.

| Resource       | Parent  | Limit |
|----------------|---------|-------|
| App            | Project | 100   |
| Integration    | Project | 100   |
| Secret         | App     | 300   |
| Dynamic Secret | App     | 10    |
| Sync           | App     | 15    |

Refer to [constraints and considerations](/hcp/docs/vault-secrets/get-started/plan-implementation/constraints-considerations) for other useful information related to usage.

To discuss increasing limits, please reach out to the [HashiCorp Support](https://support.hashicorp.com/hc/en-us/requests/new) team.
