# Retrieve secrets from Vault Secrets

You can retrieve secrets store in HCP Vault Secrets using [the HCP Portal](https://portal.cloud.hashicorp.com/), HCP
CLI, or HCP API.

## Prerequisites

- An HCP IAM user or service principal with the [HCP contributor
  role](/hcp/docs/vault-secrets/get-started/configure-user-permissions) or
  higher.
- An HCP Vault Secrets [app](/hcp/docs/vault-secrets/get-started/create-app) and
  [secret](/hcp/docs/vault-secrets/static-secrets/create-static-secret) created.
- Authenticated to the [HCP Portal](/hcp/docs/vault-secrets/get-started/access-ui) or [HCP CLI](/hcp/docs/cli/commands/auth/login)

## Retrieve secret metadata examples

<Tabs>
<Tab heading="CLI" group="cli">

The `hcp vault-secrets secrets read` command reads a static, rotating, or
dynamic secret's metadata from the Vault Secrets application.

@include 'hcp-cli/commands/vault-secrets/read-examples.mdx'

</Tab>
<Tab heading="UI" group="ui">

1. From the HCP Vault Secrets dashboard, select the app you want to view a
   secrets metadata in.

1. Click the name of the secret.

   Metadata information such as the secret type, and secret version is
   displayed. The secret value is masked.

</Tab>
</Tabs>

## Retrieve secret examples

<Tabs>
<Tab heading="CLI" group="cli">

The `hcp vault-secrets secrets open` command reads the plaintext value of a
static, rotating, or dynamic secret from the Vault Secrets application.

@include 'hcp-cli/commands/vault-secrets/open-examples.mdx'

</Tab>
<Tab heading="UI" group="ui">

1. From the HCP Vault Secrets dashboard, select the app you want to view a
   secrets metadata in.

1. Click the name of the secret.

   Metadata information such as the secret type, and secret version is
   displayed. The secret value is masked.

1. Click the view icon to reveal the secret value.

</Tab>
</Tabs>
