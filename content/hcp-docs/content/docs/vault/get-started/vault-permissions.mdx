# HCP Vault Dedicated permissions

The following table lists HCP Vault Dedicated permissions based on Role-Based Access
Control (RBAC).

| HCP Vault Dedicated Permissions |  Viewer  | Contributor |  Admin   |
| ------------------------------- | :------: | :---------: | :------: |
| Access audit logs               | &#10060; |  &#10060;   | &#9989;  |
| Access Vault UI with HCP Proxy  | &#10060; |  &#9989;    | &#9989;  |
| Cluster scaling                 | &#10060; |  &#10060;   | &#9989;  |
| Create, edit, delete clusters   | &#10060; |   &#9989;   | &#9989;  |
| Create, remove snapshots        | &#10060; |   &#9989;   | &#9989;  |
| Generate admin tokens           | &#10060; |   &#9989;   | &#9989;  |
| Lock/unlock clusters            | &#10060; |   &#9989;   | &#9989;  |
| Performance replication         | &#9989;  |   &#9989;   | &#10060; |
| View clusters                   | &#9989;  |   &#9989;   | &#9989;  |
| View snapshots                  | &#9989;  |   &#9989;   | &#9989;  |
