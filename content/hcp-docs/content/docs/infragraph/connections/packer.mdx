# HCP Packer connection

The HCP Packer connection connects your HCP Packer data to the resource graph.

## Set up the connection

To establish a new HCP Packer connection: 

1. From the Resource graph **Overview**, at the top right, click **Create Connections**. 
1. Select **HCP Packer** from the list of data sources. 
1. Click **Select Packer**. 
1. Click **Create connection**.

## Sync HCP Packer data

The HCP Packer connection automatically scans every HCP Packer registry in your HCP organization every 30 minutes, however it does not update your inventory until you manually sync the connection. When you first create the HCP Packer connection, it may take up to 30 minutes for Infragraph to populate your inventory with data from HCP Packer.

To find the current sync status for your HCP Packer data: 

1. Click **Connections** in your HCP organization's left navigation panel.
1. Check the HCP Packer connection sync status. 

    - `Awaiting initial sync` means that the connection was created but the first data sync does not initiated. 
    - `Successful` means there were no errors in the latest sync and the data was synced. 
    - `Partial failure` means at least one error occurred in the latest sync. 

To manually initiate a new sync, next to the Data source, under **Actions**, click **`…`** and then select **Sync connection**.

## View HCP Packer data

To find HCP Packer data connected to your resource graph, complete any of the following actions:

- Go to the **Inventory** page. Click an individual resource type to view it in the graph explorer. 
- Query resources with any of the existing queries that include HCP Packer resources.
- Build the query from scratch using HCP Packer resources in the query editor.
