# Connect Infragraph to data sources

Before you can view and query data, you must connect your data sources to Infragraph and run a sync operation to populate the inventory.

<Note>

During the beta release, HCP organizations are limited to one connection per connection type.

</Note>

## Overview

Infragraph supports the following connections:

External data sources:

- [AWS](/hcp/docs/infragraph/connections/aws)

Internal data sources:

- [HCP Terraform](/hcp/docs/infragraph/connections/terraform)
- [HCP Packer](/hcp/docs/infragraph/connections/packer)
