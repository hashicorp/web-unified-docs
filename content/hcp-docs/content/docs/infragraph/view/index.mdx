# Infragraph views

Infragraph includes several options for viewing and understanding your infrastructure.

## Inventory

The Inventory page provides an overview of all data retrieved by connected data sources. Use this view to explore resources and their details.

To access the Inventory page, click **Inventory** in your HCP organization's left navigation panel.

## Graph explorer

Use the graph explorer to build queries and custom filters for understanding your infrastructure. You can view the results in either a table view or a graph view. By default, the resource graph displays results as a graph showing your resources and their relationships. 

To view a list of resources with their names and data sources, click **Table** to switch to table view. You can switch between table and graph views at any time.

To learn more, refer to [Graph explorer](/hcp/docs/infragraph/view/explorer).

## Inspect resource details

In either view, you can inspect a resource's details. You can find information about the resource, including its status, applied tags, and its relationships to other resources.

In the table view, select the resource name link to view the resource details in a side panel.

In the graph view, click the node to trigger the same result.
