# hcp vault-secrets secrets versions

Command: `hcp vault-secrets secrets versions` 

The `hcp vault-secrets secrets versions` command group lets you manage a
secret's versions.

## Usage

```shell-session
$ hcp vault-secrets secrets versions <command> [Optional Flags]
```

## Commands

- [`list`](/hcp/docs/cli/commands/vault-secrets/secrets/versions/list) - List a secret's versions.
