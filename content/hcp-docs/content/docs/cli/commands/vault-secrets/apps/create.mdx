# hcp vault-secrets apps create

Command: `hcp vault-secrets apps create` 

The `hcp vault-secrets apps create` command creates a new Vault Secrets
application.

## Usage

```shell-session
$ hcp vault-secrets apps create NAME [Optional Flags]
```

## Examples

Create a new application:

```shell-session
$ hcp vault-secrets apps create company-card \
  --description "Stores corporate card info."
```

## Positional arguments

- `NAME` - The name of the app to create.

## Flags

- `--description=DESCRIPTION` - An optional description for the app to create.
