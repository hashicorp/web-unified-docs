# hcp vault-secrets apps list

Command: `hcp vault-secrets apps list` 

The `hcp vault-secrets apps list` command list all Vault Secrets applications.

## Usage

```shell-session
$ hcp vault-secrets apps list [Optional Flags]
```

## Examples

List applications:

```shell-session
$ hcp vault-secrets apps list
```
