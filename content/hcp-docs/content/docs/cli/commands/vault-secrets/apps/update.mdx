# hcp vault-secrets apps update

Command: `hcp vault-secrets apps update` 

The `hcp vault-secrets apps update` command updates the description of a Vault
Secrets application.

## Usage

```shell-session
$ hcp vault-secrets apps update NAME --description=DESCRIPTION [Optional Flags]
```

## Examples

Update an application:

```shell-session
$ hcp vault-secrets apps update company-card --description "Visa card info"
```

## Positional arguments

- `NAME` - The name of the app to update.

## Required flags

- `--description=DESCRIPTION` - The updated app description.
