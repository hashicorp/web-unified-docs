# hcp vault-secrets apps read

Command: `hcp vault-secrets apps read` 

The `hcp vault-secrets apps read` command gets a Vault Secrets application.

## Usage

```shell-session
$ hcp vault-secrets apps read NAME [Optional Flags]
```

## Examples

Read an application:

```shell-session
$ hcp vault-secrets apps read company-card
```

## Positional arguments

- `NAME` - The name of the app to read.
