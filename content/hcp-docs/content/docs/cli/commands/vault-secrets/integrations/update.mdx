# hcp vault-secrets integrations update

Command: `hcp vault-secrets integrations update` 

The `hcp vault-secrets integrations update` command updates a Vault Secrets
integration. The configuration for updating your integration will be read from
the provided HCL config file. The following fields are  required: [type
details]. For help populating the details for an integration type, please refer
to the  [API reference
documentation](https://developer.hashicorp.com/hcp/api-docs/vault-secrets/2023-11-28).

## Usage

```shell-session
$ hcp vault-secrets integrations update NAME --config-file=CONFIG_FILE [Optional
  Flags]
```

## Examples

Update a Vault Secrets integration:

```shell-session
$ hcp vault-secrets integrations update sample-integration
--config-file=path-to-file/config.hcl
```

## Positional arguments

- `NAME` - The name of the integration to update.

## Required flags

- `--config-file=CONFIG_FILE` - File path to read integration config data.
