# hcp vault-secrets integrations read

Command: `hcp vault-secrets integrations read` 

The `hcp vault-secrets integrations read` command gets a Vault Secrets
integration.

## Usage

```shell-session
$ hcp vault-secrets integrations read NAME [Optional Flags]
```

## Examples

Read an integration:

```shell-session
$ hcp vault-secrets integrations read sample-integration --type twilio
```

## Positional arguments

- `NAME` - The name of the integration to read.

## Flags

- `--type=TYPE` - The type of the integration to read.
