# hcp vault-secrets integrations create

Command: `hcp vault-secrets integrations create` 

The `hcp vault-secrets integrations create` command creates a new Vault Secrets
integration. When the `--config-file` flag is specified, the configuration for
your integration will be read from the provided HCL config file. The following
fields are required: [type details]. For help populating the details for an
integration type, please refer to the  [API reference
documentation](https://developer.hashicorp.com/hcp/api-docs/vault-secrets/2023-11-28).
When the `--config-file`  flag is not specified, you will be prompted to create
the integration interactively.

## Usage

```shell-session
$ hcp vault-secrets integrations create NAME [Optional Flags]
```

## Examples

Create a new Vault Secrets integration:

```shell-session
$ hcp vault-secrets integrations create sample-integration
--config-file=path-to-file/config.hcl
```

## Positional arguments

- `NAME` - The name of the integration to create.

## Flags

- `--config-file=CONFIG_FILE` - File path to read integration config data.
