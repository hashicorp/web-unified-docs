# hcp vault-secrets gateway-pools list

Command: `hcp vault-secrets gateway-pools list` 

The `hcp vault-secrets gateway-pools list` command lists all Vault Secrets
gateway pools.

## Usage

```shell-session
$ hcp vault-secrets gateway-pools list [Optional Flags]
```

## Examples

List gateway-pools:

```shell-session
$ hcp vault-secrets gateway-pools list
```
