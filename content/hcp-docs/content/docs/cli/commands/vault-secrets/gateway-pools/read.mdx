# hcp vault-secrets gateway-pools read

Command: `hcp vault-secrets gateway-pools read` 

The `hcp vault-secrets gateway-pools read` command gets a Vault Secrets gateway
pool.

## Usage

```shell-session
$ hcp vault-secrets gateway-pools read NAME [Optional Flags]
```

## Examples

Read a gateway pool:

```shell-session
$ hcp vault-secrets gateway-pools read company-tunnel
```

## Positional arguments

- `NAME` - The name of the gateway pool to read.
