# hcp vault-secrets gateway-pools delete

Command: `hcp vault-secrets gateway-pools delete` 

The `hcp vault-secrets gateway-pools delete` command deletes a Vault Secrets
gateway pool.

## Usage

```shell-session
$ hcp vault-secrets gateway-pools delete NAME [Optional Flags]
```

## Examples

Delete a gateway pool:

```shell-session
$ hcp vault-secrets gateway-pools delete company-tunnel
```

## Positional arguments

- `NAME` - The name of the gateway pool to delete.
