# hcp profile profiles activate

Command: `hcp profile profiles activate` 

The `hcp profile profiles activate` command activates an existing profile.

## Usage

```shell-session
$ hcp profile profiles activate NAME [Optional Flags]
```

## Examples

To active profile `my-profile`, run:

```shell-session
$ hcp profile profiles activate my-profile
```

## Positional arguments

- `NAME` - The name of the profile to activate.
