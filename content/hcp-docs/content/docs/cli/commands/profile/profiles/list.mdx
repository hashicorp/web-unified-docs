# hcp profile profiles list

Command: `hcp profile profiles list` 

The `hcp profile profiles list` command lists existing HCP profiles.

## Usage

```shell-session
$ hcp profile profiles list [Optional Flags]
```

## Examples

To list existing profiles, run:

```shell-session
$ hcp profile profiles list
```
