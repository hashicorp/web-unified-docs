# hcp profile profiles create

Command: `hcp profile profiles create` 

The `hcp profile profiles create` command creates a new named profile.

Profile names start with a letter and may contain lower case letters a-z, upper
case letters A-Z, digits 0-9, and hyphens '-'. The maximum length for a profile
name is 64 characters.

## Usage

```shell-session
$ hcp profile profiles create NAME [Optional Flags]
```

## Examples

To create a new profile, run:

```shell-session
$ hcp profile profiles create my-profile
```

## Positional arguments

- `NAME` - The name of the profile to create.

## Flags

- `--no-activate` - Disables automatic activation of the newly created profile.
