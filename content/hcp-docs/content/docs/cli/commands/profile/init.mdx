# hcp profile init

Command: `hcp profile init` 

The `hcp profile init` command configures the HCP CLI to run commands against
the correct context; namely against the desired organization and project ID.
This command is interactive. To set configuration using non-interactively prefer
using `hcp profile set`.

For a list of all available options, run `hcp config --help`.

## Usage

```shell-session
$ hcp profile init [Optional Flags]
```

## Flags

- `--vault-secrets` - Initializes Vault Secrets configuration.
