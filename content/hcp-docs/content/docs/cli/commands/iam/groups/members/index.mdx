# hcp iam groups members

Command: `hcp iam groups members` 

The `hcp iam groups members` command group lets you manage group membership.

## Usage

```shell-session
$ hcp iam groups members <command> [Optional Flags]
```

## Commands

- [`list`](/hcp/docs/cli/commands/iam/groups/members/list) - List the members of a group.
- [`add`](/hcp/docs/cli/commands/iam/groups/members/add) - Add members to a group.
- [`delete`](/hcp/docs/cli/commands/iam/groups/members/delete) - Delete a membership from a group.
