# hcp iam users

Command: `hcp iam users` 

The `hcp iam users` command group lets you manage the users of an HCP
organization.

## Usage

```shell-session
$ hcp iam users <command> [Optional Flags]
```

## Commands

- [`list`](/hcp/docs/cli/commands/iam/users/list) - List the organization's users.
- [`read`](/hcp/docs/cli/commands/iam/users/read) - Show metadata for the given user.
- [`delete`](/hcp/docs/cli/commands/iam/users/delete) - Delete a user from the organization.
