# hcp iam users delete

Command: `hcp iam users delete` 

The `hcp iam users delete` command deletes a user from the organization.

## Usage

```shell-session
$ hcp iam users delete ID [Optional Flags]
```

## Examples

Delete a user:

```shell-session
$ hcp iam users delete example-id-123
```

## Positional arguments

- `ID` - The ID of the user to delete.
