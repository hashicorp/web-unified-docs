# hcp iam roles

Command: `hcp iam roles` 

The `hcp iam roles` command group lets you interact with an HCP organization's
roles.

## Usage

```shell-session
$ hcp iam roles <command> [Optional Flags]
```

## Commands

- [`list`](/hcp/docs/cli/commands/iam/roles/list) - List an organization's roles.
