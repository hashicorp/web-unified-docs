# hcp iam roles list

Command: `hcp iam roles list` 

The `hcp iam roles list` command lists the roles that exist for an HCP
organization.

When referring to a role in an IAM binding, use the role's ID (e.g.
"roles/admin").

## Usage

```shell-session
$ hcp iam roles list [Optional Flags]
```
