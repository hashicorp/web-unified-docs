# hcp iam service-principals list

Command: `hcp iam service-principals list` 

The `hcp iam service-principals list` command lists the service principals.

To list organization service principals, set the  `--project` flag to `-`.

## Usage

```shell-session
$ hcp iam service-principals list [Optional Flags]
```
