# hcp iam service-principals keys delete

Command: `hcp iam service-principals keys delete` 

The `hcp iam service-principals keys delete` command deletes a service
principal key.

## Usage

```shell-session
$ hcp iam service-principals keys delete KEY_NAME [Optional Flags]
```

## Examples

Create a new service principal key:

```shell-session
$ hcp iam service-principals keys delete \
  iam/project/example/service-principal/example-sp/key/3KgtSLWTSs
```

## Positional arguments

- `KEY_NAME` - The resource name of the service principal key to delete.
