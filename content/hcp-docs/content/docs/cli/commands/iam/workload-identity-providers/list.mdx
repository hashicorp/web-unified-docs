# hcp iam workload-identity-providers list

Command: `hcp iam workload-identity-providers list` 

The `hcp iam workload-identity-providers list` command lists the workload
identity providers for a given service principal.

## Usage

```shell-session
$ hcp iam workload-identity-providers list SP_NAME [Optional Flags]
```

## Examples

List workload identity provider given the service principal's resource name suffix:

```shell-session
$ hcp iam workload-identity-provider list example-sp
```

List workload identity provider given the service principal's resource name:

```shell-session
$ hcp iam workload-identity-provider list \
  iam/project/example-project/service-principal/example-sp
```

## Positional arguments

- `SP_NAME` - The name of the service principal to list workload identity providers for. The
	name may be specified as either:
	
	* The service principal's resource name. Formatted as one of the following:
	    * `iam/project/PROJECT_ID/service-principal/SP_NAME`
	    * `iam/organization/ORG_ID/service-principal/SP_NAME`
	* The resource name suffix, `SP_NAME`.
