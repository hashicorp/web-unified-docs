# hcp organizations

Command: `hcp organizations` 

The `hcp organizations` command group lets you interact with an existing HCP
organization.

## Usage

```shell-session
$ hcp organizations <command> [Optional Flags]
```

## Aliases

- `orgs`. For example: `hcp orgs <command>`

## Command groups

- [`iam`](/hcp/docs/cli/commands/organizations/iam) - Manage an organization's IAM policy.

## Commands

- [`read`](/hcp/docs/cli/commands/organizations/read) - Show metadata for the organization.
- [`list`](/hcp/docs/cli/commands/organizations/list) - List organizations.
