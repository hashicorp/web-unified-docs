# hcp organizations read

Command: `hcp organizations read` 

The `hcp organizations read` command shows metadata for the organization.

## Usage

```shell-session
$ hcp organizations read ID [Optional Flags]
```

## Positional arguments

- `ID` - ID of the organization to read.
