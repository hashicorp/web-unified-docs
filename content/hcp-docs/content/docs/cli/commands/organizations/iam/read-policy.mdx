# hcp organizations iam read-policy

Command: `hcp organizations iam read-policy` 

The `hcp organizations iam read-policy` command reads the IAM policy for the
organization.

## Usage

```shell-session
$ hcp organizations iam read-policy [Optional Flags]
```

## Examples

Read the IAM Policy for the organization:

```shell-session
$ hcp organizations iam read-policy
```
