# hcp waypoint

Command: `hcp waypoint` 

The `hcp waypoint` command group lets you manage HCP Waypoint resources through
the CLI. These commands let you to interact with their HCP Waypoint instance to
manage their application deployment process.

## Usage

```shell-session
$ hcp waypoint <command> [Optional Flags]
```

## Command groups

- [`tfc-config`](/hcp/docs/cli/commands/waypoint/tfc-config) - Manage Terraform Cloud Configurations.
- [`actions`](/hcp/docs/cli/commands/waypoint/actions) - Manage action configuration options for HCP Waypoint.
- [`agent`](/hcp/docs/cli/commands/waypoint/agent) - Run and manage a Waypoint Agent.
- [`templates`](/hcp/docs/cli/commands/waypoint/templates) - Manage HCP Waypoint templates.
- [`add-ons`](/hcp/docs/cli/commands/waypoint/add-ons) - Manage HCP Waypoint add-ons and add-on definitions.
- [`applications`](/hcp/docs/cli/commands/waypoint/applications) - Manage HCP Waypoint applications.
