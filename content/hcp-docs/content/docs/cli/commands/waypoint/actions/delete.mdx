# hcp waypoint actions delete

Command: `hcp waypoint actions delete` 

The `hcp waypoint actions delete` command deletes an existing action. This will
remove the action completely from HCP Waypoint.

## Usage

```shell-session
$ hcp waypoint actions delete --name [Optional Flags]
```

## Required flags

- `-n, --name` - The name of the action to delete.
