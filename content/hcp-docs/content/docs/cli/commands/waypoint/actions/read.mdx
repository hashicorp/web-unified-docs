# hcp waypoint actions read reference

Command: `hcp waypoint actions read` 

The `hcp waypoint actions read` command returns more details about an action
configurations.

## Usage

```shell-session
$ hcp waypoint actions read --name [Optional Flags]
```

## Required flags

- `-n, --name` - The name of the action.
