# hcp waypoint tfc-config read

Command: `hcp waypoint tfc-config read` 

The `hcp waypoint tfc-config read` command returns
the TFC Organization name and a redacted form of the TFC Team token that is set
for this HCP Project. There can only be one TFC Config set for each HCP Project.

## Usage

```shell-session
$ hcp waypoint tfc-config read [Optional Flags]
```

## Examples

Retrieve the saved TFC Config from Waypoint for this HCP Project ID:

```shell-session
$ hcp waypoint tfc-config read
```
