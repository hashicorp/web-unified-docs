# hcp waypoint tfc-config delete reference

Command: `hcp waypoint tfc-config delete` 

The `hcp waypoint tfc-config delete` command deletes
the HCP Terraform organization name and team token that is set for this HCP
Project. Only one HCP Terraform configuration is allowed for each HCP Project.

## Usage

```shell-session
$ hcp waypoint tfc-config delete [Optional Flags]
```

## Examples

Delete the saved HCP Terraform configuration from Waypoint for this HCP Project ID:

```shell-session
$ hcp waypoint tfc-config delete example-org
```
