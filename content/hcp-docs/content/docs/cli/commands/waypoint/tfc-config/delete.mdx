# hcp waypoint tfc-config delete

Command: `hcp waypoint tfc-config delete` 

The `hcp waypoint tfc-config delete` command deletes
the TFC Organization name and team token that is set for this HCP
Project. Only one TFC Config is allowed for each HCP Project.

## Usage

```shell-session
$ hcp waypoint tfc-config delete [Optional Flags]
```

## Examples

Delete the saved TFC Config from Waypoint for this HCP Project ID:

```shell-session
$ hcp waypoint tfc-config delete example-org
```
