# hcp waypoint agent group list reference

Command: `hcp waypoint agent group list` 

The `hcp waypoint agent group list` command lists groups registered.

## Usage

```shell-session
$ hcp waypoint agent group list [Optional Flags]
```

## Examples

List all groups:

```shell-session
$ hcp waypoint agent group list
```
