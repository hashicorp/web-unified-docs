# hcp waypoint agent run

Command: `hcp waypoint agent run` 

The `hcp waypoint agent run` command executes a local Waypoint Agent.

## Usage

```shell-session
$ hcp waypoint agent run [Optional Flags]
```

## Flags

- `-c, --config=PATH` - Path to configuration file for agent.
