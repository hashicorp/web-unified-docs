# hcp waypoint agent group overview

Command: `hcp waypoint agent group` 

The `hcp waypoint agent group` command group manages agent groups.

## Usage

```shell-session
$ hcp waypoint agent group <command> [Optional Flags]
```

## Commands

- [`create`](/hcp/docs/cli/commands/waypoint/agent/group/create) - Create a new HCP Waypoint Agent group.
- [`list`](/hcp/docs/cli/commands/waypoint/agent/group/list) - List HCP Waypoint Agent groups.
- [`delete`](/hcp/docs/cli/commands/waypoint/agent/group/delete) - Delete a HCP Waypoint Agent group.
