# hcp waypoint agent

Command: `hcp waypoint agent` 

The `hcp waypoint agent` command group lets you run and manage a local
Waypoint agent.

Agents are used in conjunction with HCP Waypoint Actions to allow actions to run
on your own systems when initiated from HCP Waypoint.

## Usage

```shell-session
$ hcp waypoint agent <command> [Optional Flags]
```

## Command groups

- [`group`](/hcp/docs/cli/commands/waypoint/agent/group) - Manage HCP Waypoint Agent groups.

## Commands

- [`run`](/hcp/docs/cli/commands/waypoint/agent/run) - Start the Waypoint Agent.
- [`queue`](/hcp/docs/cli/commands/waypoint/agent/queue) - Queue an operation for an agent to execute.
