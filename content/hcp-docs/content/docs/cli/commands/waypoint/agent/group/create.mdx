# hcp waypoint agent group create

Command: `hcp waypoint agent group create` 

The `hcp waypoint agent group create` command creates a new Agent group.

## Usage

```shell-session
$ hcp waypoint agent group create --name=NAME [Optional Flags]
```

## Examples

Create a new group:

```shell-session
$ hcp waypoint agent group create -n='prod:us-west-2' -d='us west production access'
```

## Required flags

- `-n, --name=NAME` - Name for the new group.

## Optional flags

- `-d, --description=DESCRIPTION` - Description for the group.
