# hcp waypoint agent queue

Command: `hcp waypoint agent queue` 

The `hcp waypoint agent queue` command queues an operation for an agent to run.

## Usage

```shell-session
$ hcp waypoint agent queue --group=NAME --id=ID [Optional Flags]
```

## Required flags

- `-g, --group=NAME` - Agent group to run for operations on.

- `-i, --id=ID` - Id of the operation to run.

## Optional flags

- `--action-run=ID` - Action run to associate operation with.

- `-d, --body=JSON` - JSON to pass to operation. Use @filename to read json from a file.
