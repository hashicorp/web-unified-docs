# hcp waypoint agent group read

Command: `hcp waypoint agent group read` 

The `hcp waypoint agent group read` command reads details about a Waypoint
Agent group.

## Usage

```shell-session
$ hcp waypoint agent group read --name=NAME [Optional Flags]
```

## Examples

Read a group by name:

```shell-session
$ hcp waypoint agent group read -n='prod:us-west-2'
```

## Required flags

- `-n, --name=NAME` - Name of the group to read.
