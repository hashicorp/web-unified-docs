# hcp waypoint agent group update

Command: `hcp waypoint agent group update` 

The `hcp waypoint agent group update` command updates an existing Agent group.

## Usage

```shell-session
$ hcp waypoint agent group update --name=NAME [Optional Flags]
```

## Examples

Update a group's description:

```shell-session
$ hcp waypoint agent group update -n='prod:us-west-2' -d='updated description'
```

## Required flags

- `-n, --name=NAME` - Name of the group to update.

## Optional flags

- `-d, --description=DESCRIPTION` - New description for the group.
