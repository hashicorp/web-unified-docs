# hcp waypoint applications destroy

Command: `hcp waypoint applications destroy` 

The `hcp waypoint applications destroy` command lets you destroy an HCP
Waypoint application and its infrastructure.

## Usage

```shell-session
$ hcp waypoint applications destroy --name=NAME [Optional Flags]
```

## Examples

Destroy an HCP Waypoint application:

```shell-session
$ hcp waypoint applications destroy -n=my-application
```

## Required flags

- `-n, --name=NAME` - The name of the application to destroy.
