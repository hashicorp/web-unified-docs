# hcp waypoint templates read

Command: `hcp waypoint templates read` 

The `hcp waypoint templates read` command lets you read an existing HCP
Waypoint template.

## Usage

```shell-session
$ hcp waypoint templates read --name=NAME [Optional Flags]
```

## Examples

Read an HCP Waypoint template:

```shell-session
$ hcp waypoint templates read -n=my-template
```

## Required flags

- `-n, --name=NAME` - The name of the template.
