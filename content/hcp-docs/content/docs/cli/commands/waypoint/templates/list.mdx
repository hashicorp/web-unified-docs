# hcp waypoint templates list reference

Command: `hcp waypoint templates list` 

The `hcp waypoint templates list` command lets you list existing HCP Waypoint
templates.

## Usage

```shell-session
$ hcp waypoint templates list [Optional Flags]
```

## Examples

List all HCP Waypoint templates:

```shell-session
$ hcp waypoint templates list
```
