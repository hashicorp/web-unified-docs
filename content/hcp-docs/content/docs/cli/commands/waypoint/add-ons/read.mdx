# hcp waypoint add-ons read reference

Command: `hcp waypoint add-ons read` 

The `hcp waypoint add-ons read` command lets you read an existing HCP Waypoint
add-on.

## Usage

```shell-session
$ hcp waypoint add-ons read --name=NAME [Optional Flags]
```

## Examples

Read an HCP Waypoint add-on:

```shell-session
$ hcp waypoint add-ons read -n=my-addon
```

## Required flags

- `-n, --name=NAME` - The name of the add-on.
