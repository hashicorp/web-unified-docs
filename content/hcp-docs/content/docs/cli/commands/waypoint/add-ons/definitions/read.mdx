# hcp waypoint add-ons definitions read

Command: `hcp waypoint add-ons definitions read` 

The `hcp waypoint add-ons definitions read` command lets you read an existing
HCP Waypoint add-on definition.

## Usage

```shell-session
$ hcp waypoint add-ons definitions read --name=NAME [Optional Flags]
```

## Examples

Read an HCP Waypoint add-on definition:

```shell-session
$ hcp waypoint add-ons definitions read -n=my-addon-definition
```

## Required flags

- `-n, --name=NAME` - The name of the add-on definition.
