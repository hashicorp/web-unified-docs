# hcp waypoint add-ons definitions list reference

Command: `hcp waypoint add-ons definitions list` 

The `hcp waypoint add-ons definitions list` command lets you list all existing
HCP Waypoint add-on definitions.

## Usage

```shell-session
$ hcp waypoint add-ons definitions list [Optional Flags]
```

## Examples

List all known HCP Waypoint add-on definitions:

```shell-session
$ hcp waypoint add-ons definitions list
```
