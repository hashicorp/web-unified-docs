# hcp waypoint add-ons list reference

Command: `hcp waypoint add-ons list` 

The `hcp waypoint add-ons list` command lists HCP Waypoint add-ons. By
supplying the "name" flag, you can list add-ons for a specific application.

## Usage

```shell-session
$ hcp waypoint add-ons list [Optional Flags]
```

## Examples

List all HCP Waypoint add-ons:

```shell-session
$ hcp waypoint add-ons list
```

List HCP Waypoint add-ons for a specific application:

```shell-session
$ hcp waypoint add-ons list --application-name my-application
```

## Flags

- `--application-name=APPLICATION_NAME` - The name of the application to list add-ons for.
