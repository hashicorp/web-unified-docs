# hcp projects iam read-policy

Command: `hcp projects iam read-policy` 

The `hcp projects iam read-policy` command reads the IAM policy for a project.

## Usage

```shell-session
$ hcp projects iam read-policy [Optional Flags]
```

## Examples

Read the IAM Policy for a project:

```shell-session
$ hcp projects iam read-policy \
  --project=8647ae06-ca65-467a-b72d-edba1f908fc8
```
