# hcp projects update

Command: `hcp projects update` 

The `hcp projects update` command shows metadata for the project.

## Usage

```shell-session
$ hcp projects update [Optional Flags]
```

## Examples

Update a project's name and description:

```shell-session
$ hcp projects update --project=cd3d34d5-ceeb-493d-b004-9297365a01af \
  --name=new-name --description="updated description"
```

## Flags

- `--description=NEW_DESCRIPTION` - New description for the project.

- `--name=NEW_NAME` - New name for the project.
