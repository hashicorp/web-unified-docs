# hcp projects delete

Command: `hcp projects delete` 

The `hcp projects delete` command deletes the specified project. The project
must be empty before it can be deleted.

## Usage

```shell-session
$ hcp projects delete [Optional Flags]
```

## Examples

Delete a project:

```shell-session
$ hcp projects delete --project=cd3d34d5-ceeb-493d-b004-9297365a01af
```
