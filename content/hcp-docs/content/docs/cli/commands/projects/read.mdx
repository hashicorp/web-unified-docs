# hcp projects read

Command: `hcp projects read` 

The `hcp projects read` command shows metadata for the project.

## Usage

```shell-session
$ hcp projects read [Optional Flags]
```

## Examples

Read a project:

```shell-session
$ hcp projects read --project=cd3d34d5-ceeb-493d-b004-9297365a01af
```
