# hcp projects

Command: `hcp projects` 

The `hcp projects` command group lets you create a new HCP Project, view
existing projects, and manage access to a project.

A principal can be granted access to a project by using `hcp projects iam
add-binding`.

To view the IAM Policy for the project, run `hcp projects iam read-policy`.

To set a project as the default project for the active profile, run `hcp profile
set project_id PROJECT_ID`.

## Usage

```shell-session
$ hcp projects <command> [Optional Flags]
```

## Command groups

- [`iam`](/hcp/docs/cli/commands/projects/iam) - Manage a project's IAM policy.

## Commands

- [`create`](/hcp/docs/cli/commands/projects/create) - Create a new project.
- [`read`](/hcp/docs/cli/commands/projects/read) - Show metadata for the project.
- [`list`](/hcp/docs/cli/commands/projects/list) - List HCP projects.
- [`delete`](/hcp/docs/cli/commands/projects/delete) - Delete a project.
- [`update`](/hcp/docs/cli/commands/projects/update) - Update an existing project.
