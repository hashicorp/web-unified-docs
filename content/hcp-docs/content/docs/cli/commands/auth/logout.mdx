# hcp auth logout

Command: `hcp auth logout` 

The `hcp auth logout` command logs out to remove access to HCP.

## Usage

```shell-session
$ hcp auth logout [Optional Flags]
```

## Examples

Logout the HCP CLI:

```shell-session
$ hcp auth logout
```
