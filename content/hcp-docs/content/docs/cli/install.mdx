## Install tab autocomplete

<Note>

If you installed the CLI using one of the Linux packages, it will have
automatically installed autocomplete.

</Note>

Install autocompletion for the HCP CLI.

   ```shell-session
   $ hcp -autocomplete-install
   ```

   For the autocomplete to take effect, either create a new terminal session or
   source your terminal configuration.

To uninstall autocompletion for the HCP CLI, use `-autocomplete-uninstall`.

```shell-session
$ hcp -autocomplete-uninstall
```
