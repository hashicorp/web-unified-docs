# Setup ticketing - Jira

Automatic ticketing of detected issues gives security and developer teams
visibility into potential issues so that they can plan, track, and report work.
Vault Radar supports Jira as a ticketing tool.

## Prerequisites

- [Atlassian API Token](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)

- [Jira Base URL](https://confluence.atlassian.com/adminjiraserver/configuring-the-base-url-938847830.html)

- [Jira Project for events and alerts to be added to.](https://support.atlassian.com/jira-software-cloud/docs/create-a-new-project/)

## Create a Jira Connection

1. Select **Settings** from the side navigation. Under the **Integration** menu, click **Jira**.
   ![Nav jira](/img/docs/vault-radar/integrations/jira/jira-int-1.png)

1. Click the **+ Connection** button.
   ![Click connection](/img/docs/vault-radar/integrations/jira/jira-int-2.png)

1. Create an [Atlassian API Token](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)

1. Paste the API token into the token input.

1. Enter the email of the user used to generate the [Atlassian API Token](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/).

1. Enter the [Jira Base URL](https://confluence.atlassian.com/adminjiraserver/configuring-the-base-url-938847830.html).

1. Click **Test & Save**.
   ![Submit connection](/img/docs/vault-radar/integrations/jira/jira-int-3.png)

## Create a Jira Subscription

1. Click the **+ Subscription** button to begin the creation process.
   ![Click subscription](/img/docs/vault-radar/integrations/jira/jira-int-4.png)

1. Choose a name for the subscription as well as the saved filters you would like the subscription to be based on.

1. Choose the connection for the subscription output.

1. Enter the Jira project key.

  <Tip>

  You can find the project key `/projects/` in the URL:
  `example.atlassian.net/jira/software/c/projects/myprojectkey`.

  </Tip>

1. Enter the Jira issue type.

1. Enter the Jira assignee ID.

  <Tip>

  You can find the assignee id after selecting a user filter at the end of the URL:
  `example.atlassian.net/.../assignee=5myassigneeid6`

  </Tip>

1. Enter the priority.

1. Click **Test & Save**.
   ![Submit subscription](/img/docs/vault-radar/integrations/jira/jira-int-5.png)

Vault Radar events matching your filters will create tickets in the specified Jira project.
