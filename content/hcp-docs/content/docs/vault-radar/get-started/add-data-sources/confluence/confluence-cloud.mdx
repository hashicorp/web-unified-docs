# Add a data source - Confluence Cloud

Vault Radar allows you to connect to any organization on Confluence Cloud.

## Add Confluence Cloud

### Prerequisites

- Organization name
- Token creator email address
- [API token](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)

## Create a connection with Confluence Cloud

### Scan with HCP Cloud

1. Click **Settings**, then click **Data Sources**.

1. Select **HCP Cloud Scan** and click **Continue**.

1. Click the **Confluence Cloud** button.

  ![Add confluence SaaS datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_1.png)

1. Enter the Organization name in the **Enter your Confluence organization name** text field.

1. Enter the email address of the Confluence user that created the API token in
   the **Enter your Confluence token creator email** text field.

1. Enter the API token in the **Enter your Confluence API token** text field.

1. Click **Next**.

  ![Add confluence SaaS datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_2.png)

1. Select either the **All active spaces** or **Select spaces to monitor** radio
   button.

     Option 1 ![Add Confluence SaaS all](/img/docs/vault-radar/confluence-saas/confluence_saas_3.png)
     Option 2 ![Add Confluence SaaS selected](/img/docs/vault-radar/confluence-saas/confluence_saas_4.png)

1. Click **Finish**. Your Confluence Cloud spaces will now begin scanning.

### Scan with agent

1. Click **Settings**, then click **Data Sources**. 

1. Select **Vault Radar Agent Scan**.

1. Click the **Confluence Cloud** button.

  ![Add confluence SaaS datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_1.png)

1. Enter the organization name in the **Enter your Confluence organization name** text field.

1. Create an environment variable for the API token in the agent environment. This should be in the format `token_creator_email:api_token`.

   **Example:**

   <CodeBlockConfig hideClipboard>

   ```shell-session
   $ export CONFLUENCE_TOKEN=myemail@example.com:mytoken
   ```

   </CodeBlockConfig>

1. Enter the API token variable in the **Environment Path** text field with the `env://` prefix.

   **Example:**

   ```plaintext
   env://CONFLUENCE_TOKEN
   ```
   
1. Click **Next**.

  ![Add confluence SaaS datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_2.png)

1. Select either the **All active spaces** or **Select spaces to monitor** radio
   button.

     Option 1 ![Add Confluence SaaS all](/img/docs/vault-radar/confluence-saas/confluence_saas_3.png)
     Option 2 ![Add Confluence SaaS selected](/img/docs/vault-radar/confluence-saas/confluence_saas_4.png)

1. Click **Finish**. Your Confluence Cloud spaces will now begin scanning.
