# Set up webhooks for Confluence Cloud

Vault Radar supports real-time scanning of any newly created or updated pages in monitored spaces via [webhooks](#enabling-webhooks-for-confluence-cloud).

### Prerequisites

- Atlassian **site administrator access** to install the Forge application

## Enable webhooks for Confluence Cloud

1. Select **Settings/Data Sources**.

1. Click on the action menu for the Confluence data source you would like to enable webhook support for.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_5.png)

1. Navigate to the Atlassian Forge app.

1. On the Forge app, click **Get app**.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_6.png)

1. Click the Atlassian site dropdown.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_7.png)

1. Select the Atlassian site that you want to enable webhook support for.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_8.png)

1. Click Install.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_9.png)

1. Confluence webhooks are now enabled for the Atlassian site.
  ![Add confluence saas datasource](/img/docs/vault-radar/confluence-saas/confluence_saas_10.png)
