# Add a data source - GitHub

Vault Radar allows you to connect to any organization on [GitHub
Cloud](#add-github-cloud) and [GitHub Enterprise Server](#add-github-enterprise-server).

<Note title="Connection limit">

When you connect to GitHub as the data source, there is an upper limit of 5000
repositories that you can connect.

</Note>

## Add GitHub Cloud

### Prerequisites

- Organization
    - Navigate to your GitHub Cloud organization
    - This should be the organization string in the URL `https://github.com/ORGANIZATION`
    - This is in the upper left corner
    ![GitHub Cloud Organization](/img/docs/vault-radar/github-cloud-organization.png)

- Personal Access Token (with the right set of scopes)

## Create a connection with GitHub Cloud

1. Select **Settings/Data Sources**.

1. Click on the **GitHub Cloud** button.

1. Enter the organization in the **GitHub Organization** text field.

1. Enter the personal access token in the **Personal Access Token** text field.

  ![Configure GitHub Cloud](/img/docs/vault-radar/github-pat-scopes.png)

1. Click **Next**.

## Add GitHub Enterprise Server

### Prerequisites

- Domain name of your instance

- Organization
    - Navigate to your GitHub Enterprise Server organization
    - This should be the organization in the URL `https://github.acme.com/ORGANIZATION`
    ![GitHub Enterprise Server Organization](/img/docs/vault-radar/github-enterprise-server-organization.png)

- Personal Access Token (with the right set of scopes)

## Create a connection with GitHub Enterprise Server

1. Select **Settings/Data Sources**.

1. Click on the **GitHub Enterprise** button.

1. Enter the domain URL in the **Domain name** text field.

1. Enter the organization in the **Organization** text field.

1. Enter the personal access token in the **Personal Access Token** text field.

  ![Configure GitHub Enterprise Server](/img/docs/vault-radar/github-pat-scopes.png)

1. Click **Next**.

## Add repositories to Vault Radar

Once you establish a connection, select which repositories you
want to install by selecting a group or all.

<Note>

Vault Radar has a limit of 5000 repositories per connection. If you have more
then 5000 repositories, Vault Radar will select the 5000 repositories with the
most recent activity.

</Note>
