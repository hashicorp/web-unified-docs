# Set up alerting - PagerDuty

You can integrate with PagerDuty to subscribe events you want to monitor.

## Prerequisites

- [PagerDuty API access key API access key](https://support.pagerduty.com/docs/api-access-keys)

## Create a PagerDuty connection

1. Select **Settings** from the side navigation. Under the **Integration** menu, click **PagerDuty**.
   ![Nav pagerduty](/img/docs/vault-radar/integrations/pagerduty/pagerduty-int-1.png)

1. Click the **+ Connection** button.
   ![Click connection](/img/docs/vault-radar/integrations/pagerduty/pagerduty-int-2.png)

1. Enter a unique name for the connection.

1. Generate an API access key following the [PagerDuty docs](https://support.pagerduty.com/docs/api-access-keys).

1. Copy your API key and paste it into the API access key field.

1. Click **Test & Save**.
   ![Submit connection](/img/docs/vault-radar/integrations/pagerduty/pagerduty-int-3.png)

## Create a PagerDuty subscription

1. Click the **+ Subscription** button to begin the creation process.
   ![Click subscription](/img/docs/vault-radar/integrations/pagerduty/pagerduty-int-4.png)

1. Choose a name for the subscription as well as the saved filters you would like the subscription to be based on.

1. Choose the connection for the subscription output.

1. Copy the desired PagerDuty service API key and paste it into the **Integration Key** field. 

  <Tip>

  If you need help locating the key, you can [follow these instructions](/hcp/docs/vault-radar/get-started/set-up-alerting/pagerduty/integration-key).

  </Tip>

1. Click **Test & Save**.
   ![Submit subscription](/img/docs/vault-radar/integrations/pagerduty/pagerduty-int-5.png)

Vault Radar events matching your filters will create incidents in the specified PagerDuty service.
