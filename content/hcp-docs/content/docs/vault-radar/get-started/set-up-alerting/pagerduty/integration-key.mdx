# PagerDuty service integration key

1. Select the **Services** tab at the top of the page.
   ![Nav services](/img/docs/vault-radar/integrations/pagerduty/integration-key/pagerduty-services-tab.png)

1. Locate the service you want to integrate with and click the **More** dropdown.
   ![Click more](/img/docs/vault-radar/integrations/pagerduty/integration-key/pagerduty-services-more.png)

1. Select the **View Integrations** option.
   ![Select integrations](/img/docs/vault-radar/integrations/pagerduty/integration-key/pagerduty-view-integrations.png)

1. Locate and copy the **Integration Key** for the service.
   ![Copy key](/img/docs/vault-radar/integrations/pagerduty/integration-key/pagerduty-integration-key.png)
