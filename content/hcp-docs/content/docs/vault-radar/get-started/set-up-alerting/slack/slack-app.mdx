# Slack App

1. Visit the [Slack App creation](https://api.slack.com/apps) page.

1. Click the **Create New App** button.
   ![Create new app](/img/docs/vault-radar/integrations/slack/slack-app/slack-app-1.png)

1. Click the **From an app manifest** option.
   ![From app manifest](/img/docs/vault-radar/integrations/slack/slack-app/slack-app-2.png)

1. Select your workspace and click **Next**.
   ![Select workspace](/img/docs/vault-radar/integrations/slack/slack-app/slack-app-3.png)

1. Copy and paste the following app manifest into the **json** input:

    ```json
    {
        "display_information": {
            "name": "HashiCorp Vault Radar",
            "description": "Pipeline Security Tools",
            "background_color": "#2b2b2b",
            "long_description": "What if code security could be improved with the same techniques we use to improve code quality? Tests and code coverage reports integrated in our workflows giving us quick feedback on PRs and in our IDEs have helped us build better, faster, and more maintainable code together, and now itâ€™s time to do the same to improve security."
        },
        "features": {
            "bot_user": {
                "display_name": "HashiCorp Vault Radar",
                "always_online": false
            }
        },
        "oauth_config": {
            "scopes": {
                "bot": [
                    "chat:write",
                    "incoming-webhook"
                ]
            }
        },
        "settings": {
            "org_deploy_enabled": false,
            "socket_mode_enabled": false,
            "token_rotation_enabled": false
        }
    }
    ```

1. Click **Next**.
   ![Click next](/img/docs/vault-radar/integrations/slack/slack-app/slack-app-4.png)

1. Confirm the permissions match the following image and click **Create**.
   ![Create app](/img/docs/vault-radar/integrations/slack/slack-app/slack-app-5.png)

1. Click the **Install to Workspace** button.
   ![Install app](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-0.png)

1. Select the channel and click **Allow**.
   ![Submit channel](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-1.png)

1. Navigate to the Slack channel and click on the link of your app

   <Warning>

   If you do not click the link for the Vault Radar app and proceed with the
   remaining steps, you will receive an error message when creating the
   integration subscription in the HCP Portal.

   </Warning>

   ![Click app link](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-2.png)

1. Click on **Add this app to a channel**.
   ![Click add app](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-3.png)

1. Select the channel from the dropdown and click **Add**.
   ![Add channel](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-4.png)

1. Verify you can see the app was added.
   ![Verify app added](/img/docs/vault-radar/integrations/slack/slack-app/slack-channel-add-5.png)

You now have a Vault Radar Slack App and can begin [creating connections](/hcp/docs/vault-radar/get-started/set-up-alerting/slack/setup).
