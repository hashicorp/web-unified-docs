# version

The `version` command prints the version of this vault-radar CLI, the SHA is is
built with, and the Golang Version.

## Examples

The command returns the version of the `vault-radar` CLI tool you are running.

```shell-session
$ vault-radar version
0.4.0, git sha (6b94393cfefc9eab505ac4e066eca68dea657674), go1.20.14 arm64
```
