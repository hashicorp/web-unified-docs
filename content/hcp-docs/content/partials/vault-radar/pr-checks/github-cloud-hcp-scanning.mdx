Vault Radar accounts are monitored by the [Vault Radar Checks
App](https://github.com/apps/hashicorp-vault-radar-checks-app).

1. Install the [Vault Radar Checks App](https://github.com/apps/hashicorp-vault-radar-checks-app).
   (You must have permissions in GitHub to install the app).

Once installed, Vault Radar checks your future PRs and commits to PRs in monitored repositories.
