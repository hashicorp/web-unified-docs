[Homebrew](https://brew.sh) is a free and open source package management system
for Mac OS X. Install the official [`hcp` CLI
formula](https://github.com/hashicorp/homebrew-tap) from the terminal.

1. Install the HashiCorp tap, a repository of all of the HashiCorp Homebrew
   packages.

   ```shell-session
   $ brew tap hashicorp/tap
   ```

1. Install `hcp` with `hashicorp/tap/hcp`.

   ```shell-session
   $ brew install hashicorp/tap/hcp
   ```

   <Note>

   This installs a signed binary and is automatically updated with every new
   official release.

   </Note>

1. Run `brew upgrade` to upgrade to the latest version.

   ```shell-session
   $ brew upgrade hashicorp/tap/hcp
   ```
