To narrow the scope of user permissions, you can set a role on the project
level. To add a user to a project, you have to [invite the user](/hcp/docs/hcp/iam/users#invite-users)
to the organization first.

1. Select the target project. ![Select a project](/img/docs/hcp-core/choose-project.png)
1. Click **Access Control (IAM)** in the sidebar to navigate to the project’s role assignments list.
1. Select the username.
1. Select one or more roles to set the user’s permissions for the project. Refer to the [project role](#project-role) tables for information about the roles you can assign.
1. Click **Save**.
