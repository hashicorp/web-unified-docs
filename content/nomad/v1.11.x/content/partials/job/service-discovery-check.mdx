Optionally, define health checks with the [`check` block][service-check] to make
sure that the service catalog only returns healthy instances. Health check
configuration is the same whether you use Consul service discovery or Nomad
service discovery.

<Tabs>

<Tab heading="countdash-api">

<CodeBlockConfig lineNumbers="true" 
                 filename="countdash.nomad.hcl"
                 highlight="7-16" >

```hcl
job "countdash" {
  group "countdash-api" {
    service {
      name = "countdash-api"
      provider = "nomad"

      check {
        name     = "Countdash API ready" 
        type     = "http"
        path     = "/actuator/health"
        interval  = "5s"
        timeout   = "5s"

        check_restart {
          limit = 0
        }
      }          
    }
  }
}
```

</CodeBlockConfig>
