# Nomad Community Edition license and support

@include '../../../global/partials/version-eol-chart.mdx'

## Community Edition license

Beginning with Nomad 2.0.0, Nomad Community Edition (CE) moves from the Business
Source License (BSL / BUSL) back to the Mozilla Public License (MPL 2.0). This
means Nomad CE returns to an open source license for the 2.0.0 release and
future CE releases. We expect to update prior BSL release branches to MPL 2.0,
without formally re-releasing prior CE versions.

## Community Edition support

We provide a 2-year base CE backport policy for 2.0.0 and later releases
that delivers important security and bug-fix changes as part of an open source
release strategy. This aligns with the IBM Support Cycle 2 (SC2) model.

CE support resources include the following:

- [Creating a GitHub issue in the Nomad
  repository](https://github.com/hashicorp/nomad/issues).
- Asking in the [Nomad discussion forum](https://discuss.hashicorp.com/c/nomad).
- Searching the [Nomad documentation](/nomad/docs) or [IBM Support
  portal](https://www.ibm.com/mysupport/s/topic/0TOKe000000kYCEOA2/nomad-selfmanaged?language=en_US).
