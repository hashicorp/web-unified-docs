# Stacks deployment functions reference

Stacks supports a subset of Terraform functions in deployment configurations.

## Supported functions

The following Terraform functions are supported in Stacks deployment
configurations. Refer to the Terraform documentation for these functions for
usage details.

### Numeric functions

- [abs](/terraform/language/functions/abs) returns the absolute value of the given number.
- [ceil](/terraform/language/functions/ceil) returns the closest whole number that is greater than or equal to the given value, which may be a fraction.
- [floor](/terraform/language/functions/floor) returns the closest whole number that is less than or equal to the given value, which may be a fraction.
- [log](/terraform/language/functions/log) returns the logarithm of a given number in a given base.
- [max](/terraform/language/functions/max) takes one or more numbers and returns the greatest number from the set.
- [min](/terraform/language/functions/min) takes one or more numbers and returns the smallest number from the set.
- [parseint](/terraform/language/functions/parseint) parses the given string as a representation of an integer in the specified base and returns the resulting number.
- [pow](/terraform/language/functions/pow) calculates an exponent, by raising its first argument to the power of the second argument.

### String functions

- [chomp](/terraform/language/functions/chomp) removes newline characters at the end of a string.
- [format](/terraform/language/functions/format) produces a string by formatting a number of other values according to a specification string.
- [formatlist](/terraform/language/functions/formatlist) produces a list of strings by formatting a number of other values according to a specification string.
- [indent](/terraform/language/functions/indent) adds a specified number of spaces to the beginning of each line in a multi-line string, except for the first line.
- [join](/terraform/language/functions/join) produces a string by concatenating all of the elements of the specified list of strings with the specified separator.
- [lower](/terraform/language/functions/lower) converts all cased letters in the given string to lowercase.
- [regex](/terraform/language/functions/regex) applies a regular expression to a string and returns the matching substrings.
- [regexall](/terraform/language/functions/regexall) applies a regular expression to a string and returns a list of all matches.
- [split](/terraform/language/functions/split) produces a list by dividing a given string at all occurrences of a given separator.
- [strrev](/terraform/language/functions/strrev) reverses the characters in a string.
- [substr](/terraform/language/functions/substr) extracts a substring from a given string by offset and length.
- [title](/terraform/language/functions/title) converts the first letter of each word in the given string to uppercase.
- [trim](/terraform/language/functions/trim) removes the specified set of characters from the start and end of the given string.
- [trimprefix](/terraform/language/functions/trimprefix) removes the specified prefix from the start of the given string.
- [trimspace](/terraform/language/functions/trimspace) removes any space characters from the start and end of the given string.
- [trimsuffix](/terraform/language/functions/trimsuffix) removes the specified suffix from the end of the given string.
- [upper](/terraform/language/functions/upper) converts all cased letters in the given string to uppercase.

### Collection functions

- [coalescelist](/terraform/language/functions/coalescelist) takes any number of list arguments and returns the first one that isn't empty.
- [compact](/terraform/language/functions/compact) takes a list of strings and returns a new list with any null or empty string elements removed.
- [concat](/terraform/language/functions/concat) takes two or more lists and combines them into a single list.
- [contains](/terraform/language/functions/contains) determines whether the list, tuple, or set given in its first argument contains at least one element that is equal to the value in the second argument.
- [distinct](/terraform/language/functions/distinct) takes a list and returns a new list with any duplicate elements removed.
- [element](/terraform/language/functions/element) retrieves a single element from a list.
- [chunklist](/terraform/language/functions/chunklist) splits a single list into fixed-size chunks, returning a list of lists.
- [flatten](/terraform/language/functions/flatten) takes a list and replaces any elements that are lists with a flattened sequence of the list contents.
- [keys](/terraform/language/functions/keys) takes a map and returns a list containing the keys from that map.
- [merge](/terraform/language/functions/merge) takes an arbitrary number of maps or objects, and returns a single map or object that contains a merged set of elements from all arguments.
- [range](/terraform/language/functions/range) generates a list of numbers using a start value, a limit value, and a step value.
- [reverse](/terraform/language/functions/reverse) takes a sequence and produces a new sequence of the same length with all of the same elements as the given sequence but in reverse order.
- [setintersection](/terraform/language/functions/setintersection) takes multiple sets and produces a single set containing only the elements that all of the given sets have in common.
- [setproduct](/terraform/language/functions/setproduct) finds all of the possible combinations of elements from all of the given sets by computing the Cartesian product.
- [setsubtract](/terraform/language/functions/setsubtract) returns a new set containing the elements from the first set that are not present in the second set.
- [setunion](/terraform/language/functions/setunion) takes multiple sets and produces a single set containing the elements from all of the given sets.
- [slice](/terraform/language/functions/slice) extracts some consecutive elements from within a list.
- [sort](/terraform/language/functions/sort) takes a list of strings and returns a new list with those strings sorted lexicographically.
- [values](/terraform/language/functions/values) takes a map and returns a list containing the values of the elements in that map.
- [zipmap](/terraform/language/functions/zipmap) constructs a map from a list of keys and a corresponding list of values.

### Encoding functions

- [csvdecode](/terraform/language/functions/csvdecode) decodes a string containing CSV-formatted data and produces a list of maps representing that data.
- [jsondecode](/terraform/language/functions/jsondecode) interprets a given string as JSON, returning a representation of the result of decoding that string.
- [jsonencode](/terraform/language/functions/jsonencode) encodes a given value to a string using JSON syntax.
- [yamldecode](/terraform/language/functions/yamldecode) parses a string as a subset of YAML, and produces a representation of its value.
- [yamlencode](/terraform/language/functions/yamlencode) encodes a given value to a string using YAML 1.2 block syntax.

### Date and time functions

- [formatdate](/terraform/language/functions/formatdate) converts a timestamp into a different time format.
- [timeadd](/terraform/language/functions/timeadd) adds a duration to a timestamp, returning a new timestamp.

### Type conversion functions

- [can](/terraform/language/functions/can) evaluates the given expression and returns a boolean value indicating whether the expression produced a result without any errors.
- [try](/terraform/language/functions/try) evaluates all of its argument expressions in turn and returns the result of the first one that does not produce any errors.
