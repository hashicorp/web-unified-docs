# Upgrading to Terraform v{{ steps.get-minor-versions.outputs.next-version }}

-> **Tip:** Use the version selector to view the upgrade guides for older Terraform versions.

Terraform v{{ steps.get-minor-versions.outputs.next-version }} is a minor release in the stable Terraform v1.0 series.

Terraform v{{ steps.get-minor-versions.outputs.next-version }} honors the
[Terraform v1.0 Compatibility Promises](https://developer.hashicorp.com/terraform/language/v1-compatibility-promises).

See [the full changelog](https://github.com/hashicorp/terraform/blob/v{{ steps.get-minor-versions.outputs.next-version }}/CHANGELOG.md)
for more details. If you encounter any problems during upgrading which are not
covered this guide, please start a new topic in
[the Terraform community forum](https://discuss.hashicorp.com/c/terraform-core)
to discuss it.
