# `terraform stacks deployment-group` reference

Use the `terraform stacks deployment-group` commands to manage deployment groups in Stack configurations.

## Description

The `deployment-group` command provides operations for managing deployment groups within your Stacks configuration. Deployment groups let you organize and coordinate deployments across multiple stacks, enabling you to list existing groups, approve pending plans, rerun deployments, and monitor deployment progress.

## Usage

```
terraform stacks <global-stacks-flags> deployment-group <subcommand>
```

## Subcommands

- [`list`](/terraform/cli/commands/stacks/deployment-group/list) - List all deployment groups in the given configuration.
- [`approve-all-plans`](/terraform/cli/commands/stacks/deployment-group/approve-all-plans) - Approve all pending plans in a deployment group.
- [`rerun`](/terraform/cli/commands/stacks/deployment-group/rerun) - Rerun a deployment group.
- [`watch`](/terraform/cli/commands/stacks/deployment-group/watch) - Watch a deployment group.
