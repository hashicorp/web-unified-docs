# `terraform stacks deployment-run approve-all-plans` reference

Use the `terraform stacks deployment-run approve-all-plans` command to approve all plans for a deployment run.

## Usage

```shell-session
$ terraform stacks <global-stacks-flags> deployment-run approve-all-plans <options>
```

## Description

Approve all plans for a deployment run.

## Options

- `-deployment-run-id`: The ID of the deployment run.
    - Required.
    - String.

## Examples

The following command approves all plans for a deployment run:

```shell-session
$ terraform stacks deployment-run approve-all-plans -deployment-run-id dr-123abc
```
