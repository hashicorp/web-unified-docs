# `terraform stacks deployment-run` reference

Use the `terraform stacks deployment-run` command to manage deployment runs.

## Description

The `terraform stacks deployment-run` command provides subcommands for managing deployment runs of your Stack. You can list, approve, cancel, and monitor deployment runs using these subcommands.

## Usage

```
terraform stacks <global-stacks-flags> deployment-run <subcommand>
```

## Subcommands

- [`list`](/terraform/cli/commands/stacks/deployment-run/list) - List all deployment runs in a deployment group.
- [`approve-all-plans`](/terraform/cli/commands/stacks/deployment-run/approve-all-plans) - Approve all pending deployment steps within a deployment run.
- [`cancel`](/terraform/cli/commands/stacks/deployment-run/cancel) - Cancel a deployment run.
- [`watch`](/terraform/cli/commands/stacks/deployment-run/watch) - Watch a deployment run.
