The session manager reviews and monitors active sessions with the ability to terminate them, if needed.
They can view targets, but not modify them.
This role is primarily focused on security oversight and compliance monitoring.
Use cases could include:
- Supporting help desk
- Monitoring active connections
- Terminating suspicious sessions

We recommend the following permissions for session managers, assigned at the org or project scope level:
```shell-session
ids=*;type=session;actions=list,read,cancel
ids=*;type=target;actions=read,list
```
