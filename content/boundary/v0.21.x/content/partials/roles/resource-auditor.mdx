The Resource Auditor can view all resources, but cannot make any changes.
They are responsible for audit, reporting, and monitoring tasks where visibility is important, but modifications are not required.
Resource Auditors are frequently security engineers, compliance observers, or external auditors.

Use cases include:
- Monitoring for security issues
- Training
- Observing for compliance
- Auditing

We recommend the following permissions for Resource Auditors, assigned at the global, org, or project scope level.
