The Host Resource Manager manages the infrastructure catalog, including hosts, host catalogs, and host sets.
They are responsible for organizing and maintaining the inventory of connectable systems.
Host Resource Managers are frequently infrastructure team members or cloud administrators.

Use cases could include:
- Managing server inventories
- Organizing hosts into sets

We recommend the following permissions for Host Resource Managers, assigned at the org or project scope level.
