The User Manager manages user identities, accounts, and group memberships.
They are responsible for user lifecycle management and organizing users into functional groups.
User Managers are frequently HR or IT administrators.

Use cases include:
- Onboarding and offboarding users
- Managing access groups
- Troubleshooting accounts

We recommend the following permissions for User Managers, assigned at the global or org scope level.

Create and list users:
```shell-session
type=user;actions=create,list
```
Read, update, and delete users. Add accounts to a user, set the full set of accounts on a user, and remove accounts from a user:
```shell-session
ids=*;type=user;actions=read,update,delete,add-accounts,set-accounts,remove-accounts
```
Create and list groups:
```shell-session
type=group;actions=create,list
```
Read, update, and delete groups. Add members to a group, set the full set of members on a group, and remove members from a group:
```shell-session
ids=*;type=group;actions=read,update,delete,add-members,set-members,remove-members
```
Create, read, update, and delete accounts:
```shell-session
ids=*;type=account;actions=create,read,update,delete
```
