## Boundary 1.0.0

GA date: 2026-06-30

- **Boundary Desktop version**: [3.0.0](#boundary-desktop-3-0-0)
- **Client Agent version**: [1.0.0](#client-agent-1-0-0)
- **Boundary installer version**: [1.0.0](#boundary-installer-1-0-0)

### New features

- [RDP session recording](/boundary/docs/session-recording) - You can now configure RDP targets to record sessions.
- [SSH session recording recovery](/boundary/docs/session-recording) - A new feature automatically attempts to recover SSH session recordings if a worker loses the connection during a recording.
- Helm chart - You can now install Boundary controllers and workers in Kubernetes environments using a Helm chart. Refer to the following topics for more information:
  - [Helm chart overview](/boundary/docs/deploy)
  - [Deploying a controller](/boundary/docs/deploy/self-managed/configure-controllers)
  - [Deploying a worker](/boundary/docs/deploy/self-managed/deploy-workers)

### Bug fixes and security patches

- Updated worker connection handling to avoid CPU spikes.
- Fixed issue with email and name population in event logging.
- Fixed the visibility of the search Cancel button in dark mode.

### Improvements and behavior changes

- [Project-level aliases](/boundary/docs/targets/configuration) - You can now create aliases at the project level.
- [Grants UI update](/boundary/docs/rbac) - A new code editor featuring autocomplete and linting support was added to help create grant strings for roles.
