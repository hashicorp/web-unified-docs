The Org Admin manages a specific organization, with full control over authentication methods, user accounts, group memberships, and role assignments in that organization.
They represent the highest administrative role at the organization level.
Org Admins manage an org scope and any nested scopes.

Use cases could include:
- Managing projects within an org
- Setting up auth methods
- Managing users at an org level
- Configuring storage buckets

We recommend the following permissions for Org Admins, assigned to the org scope the administrator manages.
