# Select event rules

HCP Vault Radar event rules allow you to customize events' severity based on the secret type. 


## Edit severities

1. Select **Settings/Event Rules**.

1. Review the available event types, click the **Severity** drop-down, and select the desired severity level.

  ![Select Event Rules](/img/docs/vault-radar/select-event-rules.png)

You can also edit or delete an event rule. The updated event ruless do not impact the previous scan results but affect future scans. When you run a reconciliation scan, on-demand scan, PR webhook scan, or similar, you can view events based on the updated severities.
